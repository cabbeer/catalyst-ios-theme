CATALYST is an minimal iOS theme developed by Manik Rathee.

It supports a set of native applications and over 100 third party apps. It also comes with the date & weather widgets displayed in the screenshot to the left. The theme is a beautiful and simplistic set of icons that works best with bright and colorful backgrounds.
